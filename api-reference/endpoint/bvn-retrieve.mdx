---
title: "POST /v1/identity/bvn/retrieve"
description: "Retrieve BVN by phone number."
---

## Authentication

Required: `api.user` + `throttle:api-key`

## Headers

- `Authorization: Bearer <API_KEY>` **or** `X-API-KEY: <API_KEY>`
- `Content-Type: application/json`
- `Accept: application/json`

## Request Schema

| Field | Type | Required | Rules |
| --- | --- | --- | --- |
| `phone_number` | string | Yes | `required`, `digits:11` |
| `is_consent` | boolean/string | Yes | `required`, `accepted` |

## Sample Request

```json
{
  "phone_number": "08012345678",
  "is_consent": true
}
```

## cURL

```bash
curl -X POST "https://api.example.com/v1/identity/bvn/retrieve" \
  -H "X-API-KEY: <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number": "08012345678",
    "is_consent": true
  }'
```

## Sample Success Response (200)

```json
{
  "status": true,
  "message": "BVN retrieval successful",
  "data": {
    "type": "service-dependent object"
  }
}
```

## Sample Error Response (422)

```json
{
  "status": false,
  "message": "Validation failed.",
  "errors": {
    "phone_number": [
      "The phone number field must be 11 digits."
    ]
  }
}
```
