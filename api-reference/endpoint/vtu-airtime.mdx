---
title: "POST /v1/vtu/airtime"
description: "Purchase airtime."
---

## Authentication

Required: `api.user` + `throttle:api-key`

## Headers

- `Authorization: Bearer <API_KEY>` **or** `X-API-KEY: <API_KEY>`
- `Content-Type: application/json`
- `Accept: application/json`

## Request Schema

| Field | Type | Required | Rules |
| --- | --- | --- | --- |
| `network` | string | Yes | `required`, `string` (intended values: `mtn`, `airtel`, `glo`, `9mobile`) |
| `phone_number` | string | Yes | `required`, `string`, `regex:^\\d{11}$` |
| `amount` | number | Yes | `required`, `numeric`, `min:50` |

## Sample Request

```json
{
  "network": "mtn",
  "phone_number": "08012345678",
  "amount": 100
}
```

## cURL

```bash
curl -X POST "https://api.example.com/v1/vtu/airtime" \
  -H "X-API-KEY: <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "network": "mtn",
    "phone_number": "08012345678",
    "amount": 100
  }'
```

## Sample Success Response (200)

```json
{
  "status": true,
  "message": "Airtime purchase successful",
  "data": {
    "type": "service-dependent object"
  }
}
```

## Sample Error Response (422)

```json
{
  "status": false,
  "message": "Validation failed.",
  "errors": {
    "amount": [
      "The amount field must be at least 50."
    ]
  }
}
```
