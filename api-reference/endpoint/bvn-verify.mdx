---
title: "POST /v1/identity/bvn/verify"
description: "Verify a BVN record."
---

## Authentication

Required (`Authorization: Bearer <API_KEY>` or `X-API-KEY: <API_KEY>`)

## Request Body

- `bvn` (string, required): exactly 11 digits
- `is_consent` (boolean-like, required): must be accepted

```json
{
  "bvn": "12345678901",
  "is_consent": true
}
```

## cURL

```bash
curl -X POST "https://api.example.com/v1/identity/bvn/verify" \
  -H "Authorization: Bearer lv_vtulab_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "bvn": "12345678901",
    "is_consent": true
  }'
```

## Success Response (200)

```json
{
  "status": true,
  "message": "BVN verification successful",
  "data": {
    "note": "service-dependent object"
  }
}
```
