## Base URL

Your base URL depends on environment:

- Production: `https://api.{your-domain}`
- Development/staging (local Laravel default): `https://{your-domain}/api`

## Versioning

Current version is `v1`.

- VTU: `/v1/vtu/*`
- Identity: `/v1/identity/*`

## Health Endpoint

`GET /`

```json
{
  "name": "VTULab API",
  "status": "online",
  "version": "v1",
  "docs": "https://docs.vtulab.com"
}
```

## Products Covered

- Airtime purchase
- Data purchase
- NIN verification and retrieval
- BVN verification and retrieval
