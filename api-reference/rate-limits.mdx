---
title: "Rate Limits"
description: "throttle:api-key behavior and request-per-minute limits."
---

Protected routes apply `throttle:api-key`.

## Limits

- Authenticated API user: **60 requests/minute**
- Fallback unauthenticated context: **30 requests/minute**

## Scope

Applies to protected endpoints under:

- `/v1/vtu/*`
- `/v1/identity/*`

## Exceeded Limit

When the limit is exceeded, Laravel throttle handling returns `429 Too Many Requests`.
